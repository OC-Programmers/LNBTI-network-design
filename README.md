# LNBTI Campus Network Design (Cisco Packet Tracer)

## 📌 Overview

This project presents a complete campus network design for the Lanka Nippon BizTech Institute (LNBTI), developed using Cisco Packet Tracer as part of an Internetworking course.

The goal was to design a scalable, reliable, and secure LAN infrastructure to support academic and administrative operations.

---

## 🧠 Key Features

* VLAN-based network segmentation for different departments
* Inter-VLAN routing using a Layer 3 Switch
* Centralized DHCP server with IP address management
* Multi-floor network topology design
* Wireless (Access Points) and wired connectivity
* Structured subnetting and IP allocation

---

## 🏗️ Technologies Used

* Cisco Packet Tracer
* VLAN Configuration
* Layer 2 & Layer 3 Switching
* DHCP & IP Addressing
* Subnetting

---

## 📂 Project Files

* **NetStruct.pkt** → Full network simulation file
* **NetStruct.pdf** → Detailed project proposal and design

---

## 🚀 How to Run

1. Open the `.pkt` file using Cisco Packet Tracer
2. Switch to **Simulation Mode**
3. Use ping or PDU tools to test connectivity
